# шпаргалка по Git

* git init*
создает новый репозиторий

* git add*
добавляет в индекс ВСЕ изменения.

* git commtit -m""*
создает коммит. описание изменений.

* git checkout*
загрузка версии (комита). нужно указать первые 4 символа комита.

* git log*
перечисляет коммиты, сделанные в репозитории
